My plan to play super mario brothers on the Raspberry Pi board was to first install the Super Mario Brother program on the Raspberri Pi board. Did I use an agile model or a waterfall model?

I used a waterfall model because I planned all of it out and then tried to execute it after the plan was fully formulated. This makes for a difficult process, because you have to wait to implement or develop and changes before testing.
